# P-TRADER AI Documentation

## Project Overview

P-TRADER AI is a professional cryptocurrency trading platform consisting of:

- Flutter mobile application
- FastAPI backend
- SQLAlchemy ORM
- Alembic database migrations
- JWT authentication
- Exchange integrations (planned)
- AI trading assistant (planned)
- Arbitrage engine (planned)
